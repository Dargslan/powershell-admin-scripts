## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinWiFiSec -Scope CurrentUser
\`\`\`

## Quick Start

\`\`\`powershell
Import-Module Dargslan.WinWiFiSec
Get-WiFiSecurityAudit
Get-WiFiSecurityAudit -Json
\`\`\`

## Requirements

- PowerShell 5.1+ or 7+
- Windows 10/11 or Server 2016+
- Some functions require Administrator privileges
