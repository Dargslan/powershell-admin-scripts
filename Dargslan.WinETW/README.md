> **Windows ETW (Event Tracing) audit toolkit — active trace sessions, providers, consumer processes, and ETW tampering detection**

Part of the [Dargslan Windows Admin Toolkit](https://dargslan.com) — 120 professional PowerShell modules.

## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinETW -Scope CurrentUser
\`\`\`

## Quick Start

\`\`\`powershell
Import-Module Dargslan.WinETW
Get-ETWAudit
Get-ETWAudit -Json
\`\`\`

## Requirements

- PowerShell 5.1+ or 7+
- Windows 10/11 or Server 2016+
- Some functions require Administrator privileges
