## Installation

```powershell
Install-Module -Name Dargslan.WinVSCodeAudit -Scope CurrentUser
```

## Quick Start

```powershell
Import-Module Dargslan.WinVSCodeAudit
Get-VSCodeAudit
Get-VSCodeAudit -Json
```

## Functions

### \`Get-VSCodeAudit\`

Reports VS Code version, installed extensions, settings, workspace trust, telemetry, and resource usage.

## Requirements

- **PowerShell 5.1+** or **PowerShell 7+**
- **Windows 10/11** or **Windows Server 2016+**
- Some functions require **Administrator** privileges
