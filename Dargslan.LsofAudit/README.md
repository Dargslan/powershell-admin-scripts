# Dargslan.LsofAudit

Audit open files, handles, and network connections per process

## Installation

```powershell
Install-Module -Name Dargslan.LsofAudit -Scope CurrentUser
```

## Usage

```powershell
Import-Module Dargslan.LsofAudit
# Run the module's primary function
Get-Command -Module Dargslan.LsofAudit
```
