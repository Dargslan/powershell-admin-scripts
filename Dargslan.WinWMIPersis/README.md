## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinWMIPersis -Scope CurrentUser
\`\`\`

\`\`\`powershell
Import-Module Dargslan.WinWMIPersis
Get-WMIPersistenceAudit
Get-WMIPersistenceAudit -Json
\`\`\`

## Requirements

- PowerShell 5.1+ or 7+
