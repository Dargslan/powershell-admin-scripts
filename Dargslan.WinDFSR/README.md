Part of the [Dargslan Windows Admin Toolkit](https://dargslan.com) — 140 professional PowerShell modules.

## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinDFSR -Scope CurrentUser
\`\`\`

## Quick Start

\`\`\`powershell
Import-Module Dargslan.WinDFSR
Get-DFSRAudit
Get-DFSRAudit -Json
\`\`\`

## Requirements

- PowerShell 5.1+ or 7+
- Windows 10/11 or Server 2016+
- Some functions require Administrator privileges
