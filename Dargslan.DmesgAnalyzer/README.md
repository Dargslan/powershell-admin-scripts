# Dargslan.DmesgAnalyzer

Analyze Windows system and boot event logs

## Installation

```powershell
Install-Module -Name Dargslan.DmesgAnalyzer -Scope CurrentUser
```

## Usage

```powershell
Import-Module Dargslan.DmesgAnalyzer
# Run the module's primary function
Get-Command -Module Dargslan.DmesgAnalyzer
```
