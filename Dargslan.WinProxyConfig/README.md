## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinProxyConfig -Scope CurrentUser
\`\`\`

## Quick Start

\`\`\`powershell
Import-Module Dargslan.WinProxyConfig
Get-ProxyAudit
Get-ProxyAudit -Json
\`\`\`

## Requirements

- PowerShell 5.1+ or 7+
- Windows 10/11 or Server 2016+
- Some functions require Administrator privileges
