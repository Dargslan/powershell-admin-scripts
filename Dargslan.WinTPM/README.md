# Dargslan.WinTPM — 2026 Edition

> **Windows TPM (Trusted Platform Module) audit toolkit — TPM version, status, health, PCR values, attestation, and key storage**

Part of the [Dargslan Windows Admin Toolkit](https://dargslan.com) — 120 professional PowerShell modules.

## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinTPM -Scope CurrentUser
\`\`\`

## Quick Start

\`\`\`powershell
Import-Module Dargslan.WinTPM
Get-TPMAudit
Get-TPMAudit -Json
\`\`\`

## Requirements

- PowerShell 5.1+ or 7+
- Windows 10/11 or Server 2016+
- Some functions require Administrator privileges
