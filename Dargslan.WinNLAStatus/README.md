## Installation

\`\`\`powershell
Install-Module -Name Dargslan.WinNLAStatus -Scope CurrentUser
\`\`\`

## Quick Start

\`\`\`powershell
Import-Module Dargslan.WinNLAStatus
Get-NLAAudit
Get-NLAAudit -Json
\`\`\`

- PowerShell 5.1+ or 7+
